# Setup Guides

## Getting Started

This document provides an overview of the setup process for the Emergency Duress App. The solution is designed to be agnostic of the authentication provider and email service, allowing flexibility in implementation choices. Sample configurations are provided for Auth0 and SendGrid to help you get started quickly.
